# FTP

Script criado para enviar arquivos via FTP.

## Config.ini

Nome  | Descrição
------------- | -------------
host  | :house: Host FTP
username  | :cop: Usuário FTP
password  | :key: Senha FTP
dir	 | :floppy_disk: Diretorio Local
remoto | :computer: Diretorio FTP


## Exemplo

	[Teste]
	host = 127.0.0.1
	username = luissilva
	password = admin321
	dir = C:\FTP
	remoto = /pasta/
